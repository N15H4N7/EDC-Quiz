# EDC Quiz

EDC Quiz is a platform developed by the technical team of EDC, TIET for recruitment purposes

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install -r requirements.txt
```
